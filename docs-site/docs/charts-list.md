Save, find, and organize your charts. Search by text or hashtags, favorite often‑used charts, tag them with categories, and back up or restore your library. No setup required.

### Quick Start
1) Open transparenta.eu → “Charts”
2) Click “Create chart” to add your first chart
3) Use the search box to filter by text or #tags
4) Mark a chart as “Favorite”
5) Click “Backup” to download your library
6) Click “Restore” to import a backup

### Guided Tour

#### Home and Search
- What you see
  - Library title “Charts”; total/favorites counters
  - Search field: “Search charts…”
  - Sort by: Newest, Oldest, Title A–Z, Title Z–A, Favorites first
- How to use
  1) Type a keyword in the search box
  2) Add hashtags in the query (e.g., “#education”)
  3) Change the Sort option
- Example
  - Search “education #report” and sort by Newest

#### Entity page (context)
- What you see
  - If you came from an entity, you can build a chart using those topics
- How to use
  1) Capture a question on the entity page
  2) Move to “Charts” and build it visually
- Example
  - Turn an Education trend into a chart

#### Spending + Filters (inside chart creation)
- What you see
  - Within chart configuration: sides, years, prefixes, and more
- How to use
  1) “Create chart” → add a series
  2) Set year, side, and topic prefixes
  3) Save to view and share

#### Organize (Favorites & Categories)
- What you see
  - Favorite icon on chart cards
  - Category tag menu; tabs for All, Favorites, and categories
- How to use
  1) Click the star to toggle Favorite
  2) Add a chart to a Category
  3) Switch tabs to filter the list
- Example
  - Tag a chart “Education” and “2024”
- Media
  <!-- - ![IMG PLACEHOLDER — Favorites & categories](./images/PLACEHOLDER-charts-list-organize.png "Star + tags + tabs") -->

#### Backup / Restore
- What you see
  - “Backup” and “Restore” buttons in the library header
  - On import conflict: “Resolve import conflicts” dialog with strategies
- How to use
  1) Click “Backup” to download a JSON file
  2) Click “Restore” to import a file
  3) If conflicts appear, choose: Skip duplicates / Replace / Keep both
- Example
  - Move your library to another device
- Media

- How to use
  1) Open a chart
  2) Click “Copy Link” to share the exact view
  3) Export images for reports

### Common Tasks (recipes)

#### Find a chart by hashtag
- Steps
  1) Type a hashtag in search (e.g., “#education”)
  2) Press Enter
- Expected result
  - List shows only matching charts
- Tip
  - You can mix text and multiple hashtags
- Link
  - LINK PLACEHOLDER — Hashtag search

#### Favorite and filter
- Steps
  1) Click the star on a chart
  2) Open the “Favorites” tab
- Expected result
  - Your chart appears under Favorites
- Tip
  - Use Favorites for quick access
- Link
  - LINK PLACEHOLDER — Favorites

#### Back up your library
- Steps
  1) Click “Backup”
  2) Save the JSON file
- Expected result
  - A backup downloads to your device
- Tip
  - Keep it in cloud storage
- Link
  - LINK PLACEHOLDER — Backup

#### Restore a library
- Steps
  1) Click “Restore”
  2) Select a JSON backup
  3) If prompted, pick a conflict strategy
- Expected result
  - Charts and categories are imported
- Tip
  - “Keep both” duplicates imported charts as new
- Link
  - LINK PLACEHOLDER — Restore

### Understanding the Numbers
- Local‑first storage
  - Charts live in your browser
- Backup content
  - Includes charts and categories, not live analytics data
- Sharing
  - Use “Copy Link” on charts for the exact view

### Troubleshooting & FAQ
- “No charts yet”
  - Create your first chart; the button is visible
- “No results”
  - Change search terms or sorting
- Import fails
  - Use a valid backup made by Transparenta.eu

### Accessibility & Keyboard tips
- Keyboard
  - Use Tab/Enter to navigate search and actions
- Visuals
  - Empty and error messages explain next steps

### Privacy & Data
- Your charts live on your device
- Backups are yours; keep them safe
- Analytics only with your consent
