**Who it's for**: Users who want to browse, search, and manage charts.

**Outcomes**: Find charts quickly, organize with favorites/categories, and open or share them.

Sections:
- Search with hashtags, sorting
- Favorites and categories
- Backup/restore and create chart

Try it now

1) Open the charts library: `/charts`
2) Type `#education` in search to filter by hashtag.
3) Sort by Updated (desc), open a chart from the list.
4) Mark a chart as Favorite and observe the badge.
