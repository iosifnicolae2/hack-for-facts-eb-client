**Who it's for**: Users who want to navigate quickly.

**Outcomes**: Find entities or pages in seconds with keyboard shortcuts.

Quick search everywhere
- Press `⌘/Ctrl + K` to open the floating search dialog from any page.
- Type an entity name or CUI; use arrows + Enter to navigate.
- On mobile, use the floating action button to open search.
