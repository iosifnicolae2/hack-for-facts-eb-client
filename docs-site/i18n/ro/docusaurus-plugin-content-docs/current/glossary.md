## Entitate
Instituție publică (primărie, consiliu județean, minister etc.).

## UAT / Județ
UAT: unitate administrativ‑teritorială (municipiu/oraș/comună). Județ: nivel județean.

## Cheltuieli / Venituri
Cele două fețe ale bugetului: cheltuire vs colectare.

## Clasificare funcțională / economică
Categorii oficiale despre destinația banilor și natura economică.

## Per Capita / Total
Normalizat la populație vs sume brute.

## Serie de grafic
Linie/bară adăugată în grafic; poate fi dinamică, calculată, statică sau personalizată.
