Această secțiune documentează API‑ul Transparenta.eu. Platforma expune un API GraphQL pentru acces bogat la date și câteva endpoint‑uri REST pentru health, asistenți AI și schema MCP.

**Pentru cine**: Dezvoltatori care integrează sau script‑ează API‑ul

**Rezultate**: Trimiteți prima cerere, navigați schema și exemplele și descoperiți helper‑ele

- Endpoint GraphQL: `POST /graphql`
- GraphiQL (doar dezvoltare): `GET /graphiql`
- Documentație REST (doar dezvoltare): `GET /docs`

Ce este disponibil

- GraphQL: entități, rapoarte, linii de execuție, clasificări, analitice (heatmap, analitice entități, totale/tendințe), seturi de date și linii agregate
- REST: health probe, helper‑e AI pentru căutare și analiză buget, MCP (SDL)
