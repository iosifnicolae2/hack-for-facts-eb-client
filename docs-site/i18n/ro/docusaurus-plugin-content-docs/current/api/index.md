---
title: API – Prezentare
---

Această secțiune documentează API‑ul Transparenta.eu. Platforma expune un API GraphQL și câteva endpoint‑uri REST (sănătate, asistenți AI, MCP).

- Endpoint GraphQL: `POST /graphql`
- GraphiQL (doar dezvoltare): `GET /graphiql`
- Documentație REST (doar dezvoltare): `GET /docs`

Ce este disponibil

- GraphQL: entități, rapoarte, linii de execuție, clasificări, analitice (heatmap, analitice entități, totale/tendințe), seturi de date și linii agregate.
- REST: health probe, rutări AI pentru căutare și analiză buget, MCP (schema SDL).
