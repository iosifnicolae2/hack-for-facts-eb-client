---
title: Deep‑link & vizualizări partajabile — Ghid pentru începători
---

Partajați exact ceea ce vedeți cu un link. Transparenta.eu salvează vederea curentă (an, filtre, hartă/tabel/grafic, setări grafic) în URL, astfel încât ceilalți deschid aceeași pagină cu aceleași setări. Nu aveți nevoie de cont.

### Start rapid
1) Deschideți transparenta.eu
2) Căutați o instituție
3) Alegeți „An raportare:” și ajustați vederea
4) Copiați URL‑ul din browser
5) În Grafice: „Share Chart” → „Copy Link”
6) Trimiteți linkul
7) Destinatarul vede aceeași vedere

### Tur ghidat

#### Acasă și Căutare
- Ce vedeți: carduri („Grafice”, „Hartă”, „Entități”), câmp de căutare
- Cum folosiți: ⌘/Ctrl+K, tastați (ex.: „Sibiu”), Enter pe rezultat

#### Pagina instituției
- Ce vedeți: „An raportare:”, trend Absolut/YoY%, coloane Cheltuieli/Venituri, Analitice, Rapoarte
- Cum folosiți (partajabil): Alegeți anul → click pe un an în trend → căutați un topic (ex.: „Învățământ”) → copiați URL‑ul

#### Cheltuieli/Venituri + Filtre
- Ce vedeți: latură, Per capita/Total (în analitice), An, prefixe funcționale/economice
- Cum folosiți (partajabil): setați latura, normalizarea, anul și prefixele → copiați URL‑ul

#### Comparați (Hartă / Entități / Grafice)
- Hartă: setați topicul → copiați URL‑ul
- Entități: aplicați filtre → copiați URL‑ul sau „Export CSV”
- Grafice: „Share Chart” → „Copy Link”

### Sarcini uzuale

#### Partajați o vedere a unei instituții pentru un an
- Pași: deschideți instituția → alegeți „An raportare:” → (opțional) căutați un topic → copiați URL‑ul

#### Partajați o Hartă pentru un topic
- Pași: „Hartă” → Cheltuieli/Venituri → Per capita/Total → An → prefix funcțional → copiați URL‑ul

#### Partajați un grafic cu două orașe
- Pași: Grafice → „Creează” → adăugați două serii → „Share Chart” → „Copy Link”

#### Descărcați un CSV din clasament
- Pași: „Entități” → setați latura/normalizarea/anul/topic → „Export CSV”

### Înțelegerea numerelor
- Per capita vs Total: Per capita împarte la populație; dacă populația e 0/lipsă, valoarea devine 0
- Totaluri vs sub‑totaluri: sub‑totalurile agregă familii de coduri
- Linkurile păstrează contextul: anul și topicurile rămân în URL

### Depanare
- Linkul pare gol: comutați pe Total; alt an; prefix mai scurt
- „Copy Link” nu merge: asigurați vizibilitatea graficului și reîncercați
- Harta se deschide în altă zonă: verificați anul/topicul pentru regiunea respectivă

### Accesibilitate & scurtături
- Focus căutare: ⌘/Ctrl+K
- Selector „An raportare”: ⌘/Ctrl+;
- Editare grafice: ⌘/Ctrl+C/X/V pentru serii; ⌘/Ctrl+D duplicare

### Confidențialitate & Date
- Surse: date oficiale de execuție bugetară
- Actualizări: ani noi apar când sunt publicați
- Consimțământ: analitice/erori avansate doar cu opt‑in
