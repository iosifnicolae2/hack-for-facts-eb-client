Bun venit! Acest ghid este pentru prima utilizare. Veți înțelege ce face aplicația, conceptele de bază și calea rapidă către insight‑uri utile.

**Pentru cine**: Utilizatori noi care vor un tur ghidat

**Rezultate**: Înțelegeți instrumentele de bază și finalizați prima analiză

## Ce puteți face

- Înțelegeți o instituție cu Detalii entitate
- Comparați geografiile cu Harta
- Ierarhizați cu Analitice entități
- Construiți și partajați grafice reproductibile

## Tur de 10 minute

1. Căutați o instituție
   - ⌘/Ctrl+K sau căsuța de căutare din Acasă; alegeți „Municipiul Sibiu”
2. Citiți pagina instituției
   - Schimbați anul; comutați trendul (Absolut vs YoY%); evidențiați „Învățământ”
3. Observați tipare pe Hartă
   - Alegeți Per capita pentru comparații corecte; click pe o regiune
4. Clasamente în Analitice entități
   - Cheltuieli/Per capita, An; sortați și exportați CSV
5. Construiți un grafic
   - Grafice → Creează; adăugați o serie „Agregat anual”; salvați și copiați linkul
