## Introduction

> **Key Insight**: Traditional budgeting asks "How much money do we need?" Performance budgeting asks "What results will we achieve with this money?" The shift from inputs to outcomes transforms how you plan, execute, and report.

Romania is progressively moving toward performance-based budgeting as part of EU recommendations and public sector modernization. While not yet fully mandatory, understanding this approach positions you for the future—and makes you a more effective steward of public funds today.

### Results Chain Levels

| Level | Definition | Example | Accountability |
|-------|------------|---------|----------------|
| **Input** | Resources used | Lei, staff, equipment | Full control |
| **Activity** | Actions taken | Repairs, inspections | Full control |
| **Output** | Products/services delivered | Km repaired, permits issued | Direct control |
| **Outcome** | Changes for beneficiaries | Better road condition, faster travel | Influenced |
| **Impact** | Long-term societal change | Economic growth, safety improvement | Contributed to |

### Common Indicator Problems

| Problem | Example | Fix |
|---------|---------|-----|
| **Vague** | "Improve services" | "Reduce average permit processing time from 15 to 10 days" |
| **Not measurable** | "Better quality roads" | "% of roads rated Good or Fair by annual survey" |
| **Not achievable** | "100% citizen satisfaction" | "Increase satisfaction from 60% to 75%" |
| **Not relevant** | "Staff training hours" as outcome | Move to output; add outcome like "service quality score" |
| **Not time-bound** | "Eventually reduce waiting times" | "Reduce by 20% by December 2025" |

## Core Concepts

### Types of Indicators

```
┌─────────────────────────────────────────────────────────────┐
│                    INDICATOR TYPES                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  INPUT INDICATORS                                           │
│  ┌────────────────────────────────────────────────────┐    │
│  │ What: Resources consumed                           │    │
│  │ Examples:                                          │    │
│  │ • Budget allocated (lei)                           │    │
│  │ • Staff hours worked                               │    │
│  │ • Equipment deployed                               │    │
│  │                                                    │    │
│  │ Use: Tracking resource utilization                 │    │
│  │ Limitation: Doesn't show what was achieved        │    │
│  └────────────────────────────────────────────────────┘    │
│                                                             │
│  OUTPUT INDICATORS                                          │
│  ┌────────────────────────────────────────────────────┐    │
│  │ What: Products/services delivered                  │    │
│  │ Examples:                                          │    │
│  │ • Number of permits issued                         │    │
│  │ • Km of roads repaired                             │    │
│  │ • Students enrolled                                │    │
│  │                                                    │    │
│  │ Use: Measuring production                          │    │
│  │ Limitation: Doesn't show if results are valuable  │    │
│  └────────────────────────────────────────────────────┘    │

│  │ • Employment rate of graduates                     │    │
│  │                                                    │    │
│  │ Use: Showing actual benefit                        │    │
│  │ Limitation: Influenced by external factors        │    │
│  └────────────────────────────────────────────────────┘    │
│                                                             │
│  EFFICIENCY INDICATORS                                      │
│  ┌────────────────────────────────────────────────────┐    │
```

## Knowledge Check

Test your understanding of performance-based budgeting.

### Question 1
Which indicator type best answers "Did citizens benefit from our road maintenance program?"

- A) Input indicator (budget spent)
- B) Output indicator (km repaired)
- C) Outcome indicator (road condition improvement)
- D) Efficiency indicator (cost per km)

<details>
<summary>Show Answer</summary>

**C) Outcome indicator (road condition improvement)**

Outcome indicators measure the actual change experienced by beneficiaries. While outputs (km repaired) show what was produced, they don't prove citizens benefited. The outcome—whether road conditions actually improved—directly answers whether the program achieved its purpose.
</details>

### Question 2
A program achieved 90% of its output target but only 60% of its outcome target. What does this suggest?

- A) The program failed completely
- B) There may be a design issue—outputs aren't producing expected outcomes
- C) The outcome target was too ambitious
- D) Budget was insufficient

<details>
<summary>Show Answer</summary>

**B) There may be a design issue—outputs aren't producing expected outcomes**

When outputs are mostly achieved but outcomes lag significantly, it suggests the logical connection in the results chain may be flawed. Perhaps the activities don't actually lead to the desired outcomes, or external factors have more influence than expected. This requires program review and potential redesign.
</details>

### Question 3
What makes an indicator "SMART"?

- A) Uses sophisticated measurement technology
- B) Is Specific, Measurable, Achievable, Relevant, and Time-bound
- C) Is approved by senior management
- D) Produces smart-looking reports

<details>
<summary>Show Answer</summary>

**B) Is Specific, Measurable, Achievable, Relevant, and Time-bound**

SMART is an acronym for the five qualities good indicators should have: Specific (clearly defined), Measurable (can be quantified), Achievable (realistic), Relevant (connected to objectives), and Time-bound (has a deadline). This framework helps ensure indicators are useful for management decisions.
</details>

### Question 4
How should efficiency indicators be used?

- A) Only to compare your institution to peers
- B) To compare performance over time and against benchmarks
- C) To punish underperforming departments
- D) Only for external reporting

<details>
<summary>Show Answer</summary>

**B) To compare performance over time and against benchmarks**

Efficiency indicators (cost per output) are most valuable for tracking improvement over time within your institution AND for benchmarking against similar entities. They help identify where productivity is improving or declining and where best practices might be learned from higher performers.
</details>

## Key Takeaways

1. **Shift from inputs to outcomes** - The question is not "how much did we spend?" but "what results did we achieve?"

2. **Build results chains** - Connect every budget line to activities, outputs, and ultimately outcomes

3. **Use SMART indicators** - Vague goals produce vague results; specific targets drive performance

4. **Explain variances honestly** - When targets aren't met, provide clear root cause analysis and corrective actions

5. **Benchmark for improvement** - Compare your efficiency to peers and over time to drive continuous improvement
