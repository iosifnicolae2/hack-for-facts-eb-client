## Introduction

> **Key Insight**: Procurement failures are among the most common sources of personal liability for public officials. Good planning prevents 80% of procurement problems—and planning happens before you sign anything.

Public procurement in Romania is governed by Legea 98/2016 and HG 395/2016. As an ordonator or authorizing officer, you're personally liable for procurement decisions. This module focuses on the planning phase—where proper preparation prevents costly mistakes.

### Current Thresholds (January 2024)

| Contract Type | Direct Acquisition | Simplified | Full SEAP Procedure |
|---------------|-------------------|------------|---------------------|
| **Goods** | < 135,060 lei | 135,060 - 270,120 lei | > 270,120 lei |
| **Services** | < 135,060 lei | 135,060 - 270,120 lei | > 270,120 lei |
| **Works** | < 450,200 lei | 450,200 - 900,400 lei | > 900,400 lei |
| **Social Services** | < 450,200 lei | 450,200 - 900,400 lei | > 900,400 lei |

**EU Thresholds** (requires full EU publication):
- Goods/Services: > 5,382,000 lei (≈ €1.1M)
- Works: > 20,649,500 lei (≈ €4.2M)

### Legal vs. Illegal Division

| Legal Lot Division | Illegal Division |
|--------------------|------------------|
| By geographic location (different sites) | By time period for same need |
| By functional purpose (different CPV) | To avoid thresholds |
| By specialized requirements | To favor certain suppliers |
| By logical work packages (construction phases) | Arbitrary value splitting |

### Value Estimation Rules

```
┌─────────────────────────────────────────────────────────────┐
│                 ESTIMATED VALUE RULES                       │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  Art. 9-17, Legea 98/2016                                  │
│                                                             │
│  GENERAL RULE                                               │
│  ┌────────────────────────────────────────────────────┐    │
│  │ Estimated value = Total payable amount             │    │
│  │                   EXCLUDING VAT                     │    │
│  │                   INCLUDING all options, renewals   │    │
│  │                   INCLUDING all lots               │    │
│  └────────────────────────────────────────────────────┘    │
│                                                             │
│  SPECIFIC RULES BY CONTRACT TYPE                            │
│                                                             │
│  Goods (recurring needs):                                   │
│  ┌────────────────────────────────────────────────────┐    │
│  │ Value = Total purchases of same type over 12 months│    │
│  │ OR previous fiscal year actual + adjustments       │    │
│  └────────────────────────────────────────────────────┘    │
│                                                             │
│  Services (fixed duration):                                 │
│  ┌────────────────────────────────────────────────────┐    │
│  │ Value = Monthly fee × Contract duration            │    │
│  │ + All options and renewals                         │    │
│  └────────────────────────────────────────────────────┘    │
│                                                             │
│  Works:                                                     │
│  ┌────────────────────────────────────────────────────┐    │
│  │ Value = Total cost of works                        │    │
│  │ + Materials supplied by authority (if any)        │    │
│  │ + All lots/phases combined                         │    │
│  └────────────────────────────────────────────────────┘    │
│                                                             │
│  Framework agreements:                                      │
│  ┌────────────────────────────────────────────────────┐    │
│  │ Value = Maximum anticipated over agreement term    │    │
│  │ (up to 4 years for goods/services, 8 for works)   │    │
│  └────────────────────────────────────────────────────┘    │
│                                                             │
│  ⚠️ Never underestimate to avoid thresholds!               │
│     ANAP can impose sanctions up to 2% of contract value   │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

## Knowledge Check

Test your understanding of procurement planning.

### Question 1
Your institution needs office furniture worth 200,000 lei (excluding VAT). Which procedure applies?

- A) Direct acquisition (achiziție directă)
- B) Simplified procedure (procedură simplificată)
- C) Open tender (licitație deschisă)
- D) Competitive dialogue

<details>
<summary>Show Answer</summary>

**B) Simplified procedure (procedură simplificată)**

Office furniture = Goods (produse). The threshold for goods is:
- < 135,060 lei: Direct acquisition
- 135,060 - 270,120 lei: Simplified procedure
- 270,120 lei: Full procedure

200,000 lei falls in the simplified procedure range.
</details>

### Question 2
You plan to buy cleaning services for 150,000 lei in Q1 and another 150,000 lei in Q3 for the same building. How should you treat this in the PAP?

- A) Two separate entries, each using simplified procedure
- B) One entry for 300,000 lei using open tender
- C) Two entries with different CPV codes
- D) Not required in PAP if under threshold individually

<details>
<summary>Show Answer</summary>

**B) One entry for 300,000 lei using open tender**

Services of the same type (same CPV code) for the same location over a 12-month period must be aggregated for threshold calculation. 300,000 lei exceeds the 270,120 lei threshold for services, so an open tender procedure is required. Splitting by time period is illegal artificial division.
</details>

### Question 3
Which of the following is a LEGAL reason to divide a procurement into lots?

- A) Each lot stays under the simplified procedure threshold
- B) Different geographic locations requiring different suppliers
- C) Breaking up work to spread payments across budget years
- D) Allowing a preferred supplier to win at least one lot

<details>
<summary>Show Answer</summary>

**B) Different geographic locations requiring different suppliers**

Dividing by geographic location is legal when it reflects genuine operational differences (e.g., buildings in different cities). Options A, C, and D describe illegal practices: threshold avoidance, arbitrary time splitting, and discrimination.
</details>

### Question 4
When must the PAP (Annual Procurement Program) be approved?

- A) By December 31 of the previous year
- B) Within 30 days of budget approval
- C) Before the first procurement procedure starts
- D) By January 15 of the current year

<details>
<summary>Show Answer</summary>

**B) Within 30 days of budget approval**

Per HG 395/2016, the PAP must be approved by the ordonator within 30 days of budget approval. Since budget approval timing varies (state budget typically late December, local budgets may be later), the deadline is tied to budget approval, not the calendar.
</details>

---

## Key Takeaways

1. **PAP is mandatory and public** - Must be approved within 30 days of budget and published

2. **Thresholds determine procedure** - Know current values (updated regularly) and use total annual needs for calculation

3. **CPV codes matter** - Main CPV determines the category (goods/services/works) for threshold purposes

4. **Lot division rules are strict** - Divide by type/location, never by time or to avoid thresholds

5. **Estimation must be realistic** - Document your methodology; underestimation to avoid procedures is illegal
