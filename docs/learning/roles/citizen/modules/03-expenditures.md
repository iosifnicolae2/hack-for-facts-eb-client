## Technical Notes for Implementation

### Components Needed

1. **BudgetAllocatorGame** - Slider-based allocation with comparison
2. **SpendingTreemap** - Interactive D3/Recharts treemap
3. **ExecutionRateChart** - Horizontal bar chart with color coding
4. **EUComparisonChart** - Spending comparison across countries
5. **ClassificationToggle** - Switch between functional/economic views
6. **CodeLookup** - Quick reference for classification codes

### Data Requirements

- Functional spending breakdown (API: `/api/analytics/spending-by-function`)
- Economic spending breakdown (API: `/api/analytics/spending-by-economic`)
- Execution rates by category (API: `/api/analytics/execution-rates`)
- EU comparison data (Eurostat)

### Accessibility

- Treemap has keyboard navigation
- Color coding supplemented with patterns/icons
- Screen reader descriptions for charts
- Text alternatives for all visual data
