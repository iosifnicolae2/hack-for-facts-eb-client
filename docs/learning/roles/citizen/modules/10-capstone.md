## Technical Notes

### Components Needed

1. **CapstoneWizard** - Multi-step guided project flow
2. **DataCollectionForm** - Input fields for budget metrics
3. **RedFlagChecker** - Automated evaluation based on inputs
4. **QuestionGenerator** - Context-aware question suggestions
5. **ReportBuilder** - Formatted PDF/HTML report generator
6. **CertificateGenerator** - Personalized completion certificate
7. **ProgressTracker** - Step completion status

### Data Integration

- Entity search (API: `/api/entities/search`)
- Entity details (API: `/api/entities/{cui}`)
- Budget data (API: `/api/analytics/entity-budget`)
- Comparison data (API: `/api/analytics/compare`)

### Gamification Elements

- Progress badges for each step
- Shareable certificate
- Leaderboard of completed analyses (optional)
- Community showcase of findings
