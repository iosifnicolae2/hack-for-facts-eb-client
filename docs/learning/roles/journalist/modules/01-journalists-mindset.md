## Introduction

### The Investigative Advantage

> **Key Insight:** Politicians announce; budgets reveal. The gap between press releases and budget lines is where stories live.

Most journalists stop at the press release. Budget documents are public but underutilized. Your competitive advantage is the willingness to dig into documents others ignore.

4. **InvestigationFunnel** - Visual diagram component

### Data Requirements

- Execution rates by ministry/category (API: `/api/analytics/execution-summary`)
- Year-over-year comparison data
- Pre-populated examples for Story Gap Finder
- Ethics checklist templates

### API Endpoints

- `GET /api/entities/{id}/execution` - Execution rates
- `GET /api/analytics/compare?years=2023,2024` - YoY comparison
- `GET /api/analytics/anomalies` - Statistical outliers

- All card flips keyboard-navigable (Enter/Space)
- Screen reader announces card state (front/back)
- Form inputs properly labeled
- Quiz answers announced on selection
