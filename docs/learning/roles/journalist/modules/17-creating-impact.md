## Platform Integration: Guided Activity

### Activity: Plan Your Next Investigation for Impact

**Step 1: Choose a Topic with Accountability Potential**

Consider:
- Is there an institution that can act on findings?
- Are there upcoming moments (elections, audits, budget cycles) that create pressure?
- Is there a coalition of stakeholders who care about this issue?

**Step 2: Complete the Impact Planning Canvas**

For your chosen topic:
- Define desired outcomes at each level
- Map the key decision-makers
- Identify the best timing
- Plan your follow-up schedule

**Step 3: Build Your Ecosystem**

Who do you need to make this investigation impactful?
- Which other journalists might collaborate or amplify?
- Which NGOs are working on this issue?
- Which officials can take action?
- Which experts can provide context?

**Step 4: Set Up Tracking**

Create a system to:
- Set reminders for follow-up dates
- Track media pickup
- Document institutional responses
- Measure outcomes over time
