# AGENTS

## Type checks

Run ```yarn typecheck``` to check for type errors.

## Naming conventions

Use human readable naming for variable, functions, etc, trying to make it easy to read and related to the context of the problem or domain.

- Use `camelCase` for variable names.
- Use `PascalCase` for component names.
- Use `kebab-case` for file names.
- Use `UPPER_CASE` for constants.
