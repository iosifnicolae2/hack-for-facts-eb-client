# Transparenta.eu Features

## Concept Introduction

Official government portals contain raw data, but navigating them requires expertise. Transparenta.eu aggregates **80+ million budget records** into one searchable, visual platform designed for citizens, journalists, and officials alike.

Instead of downloading spreadsheets and building your own analysis, you can explore budget data through interactive visualizations, compare entities side-by-side, and — most importantly — set up automated alerts so you don't have to check manually.

> **Key Insight:** Transparenta.eu turns raw government data into actionable insights. Set up alerts once, and let the platform notify you when things change.

## Core Explanation

### Feature Quick Reference

| Feature | Best For | Key Actions |
|---------|----------|-------------|
| **Budget Explorer** | Big picture understanding | Toggle views, filter by year, compare categories |
| **Entity Analytics** | Institution deep-dives | Search, compare entities, export data |
| **Map** | Geographic patterns | Click regions, view spending heatmaps |
| **Alerts** | Automated monitoring | Set conditions, get notified |

### Why Alerts Matter

Checking budget data manually is tedious and easy to forget. Alerts solve this by:

1. **Monitoring continuously** — The platform watches even when you're not looking
2. **Notifying proactively** — Get an email when something changes
3. **Filtering noise** — Only alert on conditions you care about

**Recommended first alert:** Monitor your local city/commune for new data. You'll be notified whenever Transparenta.eu updates their budget information.
