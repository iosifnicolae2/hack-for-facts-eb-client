# Raportează și acționează

## Introducerea conceptului

Ai găsit entitatea, ai colectat datele și ai identificat semnalele de alarmă. Acum este momentul să transformi analiza în acțiune. În această ultimă lecție, vei crea un raport care poate fi distribuit și vei planifica pașii concreți următori.

Aici cunoașterea devine putere. O analiză bine documentată cu întrebări clare este fundamentul implicării civice eficiente.

> **Idee cheie:** Datele fără acțiune nu schimbă nimic. Un cetățean care pune întrebări informate poate schimba totul.

### Specificația componentei: QuestionBuilder

```typescript
/**
 * Componenta QuestionBuilder
 *
 * SCOP: Generarea întrebărilor contextuale pe baza semnalelor de alarmă găsite,
 * permițând utilizatorilor să personalizeze și să salveze lista de întrebări.
 */
interface QuestionBuilderProps {
  readonly redFlagResults: RedFlagResults
  readonly entityName: string
  readonly onQuestionsComplete: (questions: GeneratedQuestions) => void
}

interface QuestionTemplate {
  readonly id: string
  readonly triggerRedFlag: string
  readonly questionText: string
  readonly placeholders: ReadonlyArray<{
    key: string
    description: string
  }>
}

interface GeneratedQuestions {
  readonly suggested: ReadonlyArray<{
    readonly template: string
    readonly filled: string
    readonly isSelected: boolean
  }>
  readonly custom: ReadonlyArray<string>
}

/**
 * GESTIONAREA STĂRII:
 * - suggestedQuestions: QuestionTemplate[] (filtrate după semnalele de alarmă)
 * - selectedQuestions: Set<string>
 * - customQuestions: string[]
 * - isComplete: boolean
 *
 * TRANZIȚII DE STARE:
 * - Componenta se încarcă -> filtrează șabloanele după semnalele detectate
 * - Utilizatorul comută întrebarea -> adaugă/elimină din setul selectat
 * - Utilizatorul scrie personalizat -> adaugă în array-ul de întrebări personalizate
 * - Cel puțin 1 întrebare selectată -> activează finalizarea
 *
 * INTERACȚIUNI UTILIZATOR:
 * - Căsuță de bifare pentru selectarea/deselectarea întrebărilor sugerate
 * - Câmp text pentru întrebări personalizate
 * - Buton „Adaugă alta" pentru întrebări personalizate suplimentare
 * - Tragere pentru reordonare (opțional)
 * - Buton „Copiază tot" pentru clipboard
 *
 * STĂRI VIZUALE:
 * - Întrebare sugerată selectată: evidențiată cu bifă
 * - Câmp personalizat activ: contur de focalizare, text placeholder
 * - Finalizat: toate întrebările vizibile, opțiuni de export afișate
 *
 * ANIMAȚII:
 * - Selectare întrebare: umplere căsuță + evidențiere subtilă (150ms)
 * - Întrebare personalizată nouă: intrare prin alunecare în jos (200ms)
 * - Confirmare copiere: flash cu bifă (300ms)
 *
 * ACCESIBILITATE:
 * - role="group" pentru categoriile de întrebări
 * - Căsuțele au aria-describedby pentru textul întrebării
 * - Câmpurile personalizate au etichete clare
 * - Anunțare actualizări număr întrebări
 */
```

### Specificația componentei: ReportBuilder

```typescript
/**
 * Componenta ReportBuilder
 *
 * SCOP: Compilarea tuturor datelor colectate, semnalelor de alarmă și întrebărilor
 * într-un document de raport formatat, care poate fi distribuit.
 */
interface ReportBuilderProps {
  readonly entityData: SelectedEntity
  readonly budgetData: CollectedBudgetData
  readonly redFlagResults: RedFlagResults
  readonly questions: GeneratedQuestions
  readonly onReportGenerated: (report: GeneratedReport) => void
}

interface GeneratedReport {
  readonly id: string
  readonly createdAt: Date
  readonly entity: SelectedEntity
  readonly summary: {
    readonly totalBudget: number
    readonly population: number
    readonly perCapita: number
  }
  readonly findings: {
    readonly overallExecution: number
    readonly investmentExecution: number
    readonly personnelShare: number
    readonly topFunctions: ReadonlyArray<{ name: string; percentage: number }>
  }
  readonly redFlags: ReadonlyArray<{
    readonly severity: 'warning' | 'red_flag'
    readonly description: string
  }>
  readonly questions: ReadonlyArray<string>
  readonly actions: ReadonlyArray<{
    readonly type: 'foia' | 'meeting' | 'alert' | 'share'
    readonly description: string
    readonly isChecked: boolean
  }>
}

/**
 * GESTIONAREA STĂRII:
 * - reportData: GeneratedReport (compilat din props)
 * - editMode: boolean
 * - selectedActions: Set<string>
 * - exportFormat: 'pdf' | 'html' | 'link'
 *
 * TRANZIȚII DE STARE:
 * - Componenta se montează -> compilează raportul din toate props-urile
 * - Utilizatorul editează câmpul -> actualizează reportData
 * - Utilizatorul bifează acțiunea -> adaugă în selectedActions
 * - Utilizatorul apasă export -> generează în formatul selectat
 *
 * INTERACȚIUNI UTILIZATOR:
 * - Click pe câmp pentru editare (editare inline)
 * - Căsuță de bifare pentru fiecare acțiune recomandată
 * - Butoane de export: PDF, Distribuie link, Tipărește
 * - Comutator „Editare" pentru modul vizualizare vs. editare
 *
 * STĂRI VIZUALE:
 * - Mod vizualizare: layout curat, pregătit pentru tipărire
 * - Mod editare: câmpurile au indicatori de editare
 * - În curs de export: spinner de încărcare pe buton
 * - Exportat: confirmare de succes cu copiere în clipboard
 *
 * ANIMAȚII:
 * - Editare câmp: fade-in contur (150ms)
 * - Succes export: animație bifă (300ms)
 * - Link de distribuire copiat: tooltip cu alunecare în sus (200ms)
 *
 * ACCESIBILITATE:
 * - role="document" pe containerul raportului
 * - Secțiunile au ierarhie corectă de titluri
 * - Butoanele de export au aria-labels clare
 * - Stiluri de tipărire pentru accesibilitate
 */
```

**Integrare date:**
- Stocare raport: `POST /api/reports` (opțional, pentru persistență)
- Generare link de distribuire: Creează URL unic de distribuire

## Planul tău de acțiune

Acum că ai raportul, ce vei face cu el?

### Acțiuni imediate

| Acțiune | Cum | Când |
|---------|-----|------|
| **Configurează alerte** | Folosește Transparenta.eu pentru a monitoriza această entitate | Astăzi |
| **Distribuie vecinilor** | Trimite raportul prietenilor implicați civic | Săptămâna aceasta |
| **Adaugă entitatea la favorite** | Acces ușor pentru verificări viitoare | Astăzi |

### Acțiuni pe termen mediu

| Acțiune | Cum | Când |
|---------|-----|------|
| **Trimite cerere L544** | Folosește întrebările din analiza ta | În 30 de zile |
| **Participă la ședința consiliului** | Găsește data următoarei ședințe publice | Următoarea sesiune |
| **Conectează-te cu grupuri locale** | Găsește organizații civice din zona ta | Luna aceasta |

### Implicare pe termen lung

| Acțiune | Cum | Când |
|---------|-----|------|
| **Urmărește an de an** | Compară datele anului viitor cu această referință | Anual |
| **Împărtășește ce ai învățat** | Învață-i pe alții să facă aceeași analiză | Continuu |
| **Votează informat** | Folosește cunoștințele bugetare în alegerile locale | La alegeri |

## Ideea cheie

> **Ai reușit!** Ai finalizat o analiză bugetară completă a administrației tale locale. Ai date, semnale de alarmă, întrebări și un plan de acțiune. Cel mai important pas acum este cel mai simplu: **fă o acțiune astăzi.** Configurează o alertă. Distribuie raportul. Participă la o ședință. Supravegherea bugetară începe cu un singur cetățean informat — și acesta ești acum tu.
