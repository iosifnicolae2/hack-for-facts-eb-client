## Introduction

### Every Lei Has a Code

Budget classification is the language of public finance. Every line in your budget has three dimensions that together answer: WHO is spending (organizational), FOR WHAT purpose (functional), and WHAT TYPE of spending (economic).

> **Key Insight:** Correct classification isn't just bureaucracy—it determines legality. Spending money under the wrong code is a compliance violation that can trigger audit findings and personal liability.

### Suggested Implementation

```typescript
interface BudgetCodeDecoder {
  functionalCode: {
    category: string       // e.g., "65"
    budgetType: string     // e.g., "02" (local)
    subcategory: string    // e.g., "04"
    detail: string         // e.g., "01"
    labels: {
      ro: string
      en: string
    }
  }
  economicCode: {
    titlu: string          // e.g., "20"
    articol: string        // e.g., "01"
    alineat: string        // e.g., "01"
    labels: {
      ro: string
      en: string
    }
  }
  plainLanguage: string
}

interface CodeDecoder {
  decode: (code: string) => BudgetCodeDecoder
  validate: (code: string) => { valid: boolean; errors: string[] }
}
```
