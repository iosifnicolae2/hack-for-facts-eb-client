# Legal Framework Mastery

Navigate the legal architecture governing public budgets.

## Primary legislation hierarchy

1. **Constituția României** (Art. 137-139)
2. **Legea 500/2002** — Public Finance (State budget framework)
3. **Legea 273/2006** — Local Public Finance (Local budget framework)
4. **Legea 98/2016** — Public Procurement
5. **Legea 544/2001** — Transparency (FOIA)
6. **OG 119/1999** — Internal control

## Key legal instruments

| Law/Regulation | Governs | Key provisions |
|----------------|---------|----------------|
| **Legea 500/2002** | State budget | Budget cycle, credit types, execution |
| **Legea 273/2006** | Local budgets | Local autonomy, transfers, borrowing |
| **Legea 98/2016** | Public procurement | Procedures, thresholds, exceptions |
| **OMFP 1792/2002** | Commitment control | ALOP phases, CFPP visa |

> **Important:** Note that OG 119/1999 is a Government Ordinance (OG), not an Emergency Ordinance (OUG).
