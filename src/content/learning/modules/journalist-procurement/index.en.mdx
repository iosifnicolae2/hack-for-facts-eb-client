By the end of this module, you will be able to:

## Platform Integration: Guided Activity

### Activity: Budget-to-Contract Investigation

**Step 1: Select Investigation Target**

In Transparenta.eu, find an entity with significant capital spending:
```
/budget-explorer?accountCategory=ch&economicCode=71
```

**Step 2: Note Budget Details**

Record:
- Entity CUI and name
- Budget amount for code 71 (capital)
- Any specific budget line descriptions

**Step 3: Search SICAP**

Go to e-licitatie.ro (SICAP) and search:
- Filter by contracting authority (using CUI)
- Filter by year matching budget
- Look at works contracts (for code 71)

**Step 4: Calculate Coverage**

- Sum all SICAP contract values
- Divide by budget allocation
- Flag if &lt;70% coverage

**Step 5: Investigate Gaps**

For unmatched amounts:
- Request direct award list via FOIA
- Check for patterns suggesting splitting
- Verify vendor legitimacy
