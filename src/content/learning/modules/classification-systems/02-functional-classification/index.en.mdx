# Functional Classification

## Concept Introduction

Functional classification answers the question: **"What is this money FOR?"** It groups all spending by PURPOSE, regardless of which institution spends it or what type of expense it is.

Romania's functional classification is **adapted from** the international COFOG (Classification of Functions of Government) standard. While the international COFOG uses codes 01-10, Romania uses a national coding system (primarily codes 50-87) organized into five functional "Parts" (PÄƒrÈ›i). This Romanian system **maps to COFOG** for Eurostat reporting, enabling international comparisons.

> **Key Insight:** Functional codes reveal PRIORITIES. If a government says education is important but code 65 gets only 3% of the budget, actions speak louder than words.

### The .01 vs .02 Pattern

This is crucial for Romania:
- **.01** = Central government level (Ministry budget)
- **.02** = Local government level (City/County budget)

So `65.01` is education funded from the Ministry of Education's central budget, while `65.02` is education funded from local (municipal/county) budgets.

### Quick Reference: Key Codes to Remember

| Part | Codes | Focus Area |
|------|-------|------------|
| **I** | 51, 53, 54, 55 | Government operations, public admin |
| **II** | 60, 61 | Defense and public order |
| **III** | 65, 66, 67, 68 | Social services (education, health, culture, welfare) |
| **IV** | 70, 74 | Housing and environment |
| **V** | 80, 84, 87 | Economic actions and transport |

> **Note:** Romania uses specific non-sequential codes, not continuous ranges. Codes like 52, 62-64, 71-73, etc. are **not defined**.
