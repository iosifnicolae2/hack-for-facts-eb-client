# Functional Classification

## Concept Introduction

Functional classification answers the question: **"What is this money FOR?"** It groups all spending by PURPOSE, regardless of which institution spends it or what type of expense it is.

Romania uses the international COFOG (Classification of Functions of Government) standard, which means you can compare Romanian spending on education directly with French or German spending â€” same codes, same meaning.

> **Key Insight:** Functional codes reveal PRIORITIES. If a government says education is important but code 65 gets only 3% of the budget, actions speak louder than words.

## Core Explanation

### Main Functional Categories

| Code | Function | What it covers |
|------|----------|----------------|
| **50** | General Public Services | Government operations, public administration |
| **51** | Defense | Military, national security |
| **54** | Public Order & Safety | Police, courts, prisons, fire services |
| **55** | Economic Affairs | Transport, agriculture, energy subsidies |
| **56** | Environment | Waste management, pollution control |
| **65** | Education | Schools, universities, training |
| **66** | Health | Hospitals, clinics, public health |
| **67** | Culture, Recreation | Museums, sports, religious affairs |
| **68** | Social Protection | Pensions, social assistance, unemployment |

### Understanding Code Structure

Functional codes have a hierarchical structure:

### The .01 vs .02 Pattern

This is crucial for Romania:
- **.01** = Central government level (Ministry budget)
- **.02** = Local government level (City/County budget)

So `65.01` is education funded from the Ministry of Education's central budget, while `65.02` is education funded from local (municipal/county) budgets.

### Quick Reference: Functions at a Glance

```
50-59 = GOVERNMENT OPERATIONS (administration, defense, order)
60-69 = SOCIAL SERVICES (education, health, culture)
70-79 = ECONOMIC & INFRASTRUCTURE (housing, environment)
80-89 = FINANCIAL & OTHER
```
