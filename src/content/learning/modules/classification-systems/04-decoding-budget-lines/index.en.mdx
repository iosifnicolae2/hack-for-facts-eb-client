# Decoding Budget Lines

## Concept Introduction

Now that you know functional (what for?) and economic (what type?) classifications, it's time to put them together. Real budget lines combine BOTH systems, creating codes like `65.02.04.01 - 71.01.01`.

At first glance, this looks like encrypted gibberish. But with your new knowledge, you can decode it instantly.

> **Key Insight:** Once you can decode a budget line, budget documents become readable maps instead of intimidating spreadsheets.

### Quick Reference Card

Based on **OMFP nr. 1954/2005** (Order of the Minister of Public Finance), Romanian budget classifications are organized into five parts:

| Part | Code | Category |
|------|------|----------|
| I. General Public Services | 51 | Public authorities |

| IV. Housing & Environment | 70 | Housing & public development |
| | 74 | Environment protection |
| V. Economic Actions | 80-87 | Economic affairs |

**Economic Classifications (What type of expense?)**

| Code | Type |
|------|------|
| 10 | Personnel |
| 20 | Goods & services |
| 30 | Interest |
| 40 | Subsidies |
| 50 | Transfers |
| 70 | Capital expenses |
| 71 | **Investments** âš ï¸ |
| 79 | Financial operations |

**Capital Code Distinction:**
- **71.01** = Acquisition of NEW fixed assets
- **71.03** = Major repairs to EXISTING assets

**Budget Level Suffixes:**
- `.01` = Central/state budget
- `.02` = Local budget
- `.03` = Social insurance
- `.05` = Health insurance
