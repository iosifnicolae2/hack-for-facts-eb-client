# Budget Classification System

Master the classification system for accurate planning and reporting.

## The three-dimensional classification

| Dimension | Question | Example |
|-----------|----------|---------|
| **Organizational** | WHO spends? | Ministerul Educației → ISJ Sibiu → Liceul X |
| **Functional** | FOR WHAT purpose? | 65.02.04.01 = Secondary education |
| **Economic** | WHAT TYPE of spending? | 71.01.01 = Construction |

## Economic classification structure

- **TITLU 10** — Personnel costs (Salaries, contributions)
- **TITLU 20** — Goods and services (Utilities, repairs, etc.)
- **TITLU 71** — Capital investments (Construction, equipment)
- **TITLU 81** — Loan repayments
