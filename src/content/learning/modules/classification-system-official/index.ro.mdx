# Sistemul de Clasificare Bugetară

Stăpânește sistemul de clasificare pentru o planificare și raportare precisă.

## Clasificarea tridimensională

| Dimensiune | Întrebare | Exemplu |
|-----------|----------|---------|
| **Organizational** | CINE cheltuiește? | Ministerul Educației → ISJ Sibiu → Liceul X |
| **Functional** | PENTRU CE scop? | 65.02.04.01 = Învățământ secundar |
| **Economic** | CE TIP de cheltuială? | 71.01.01 = Construcții |

## Structura clasificării economice

- **TITLU 10** — Cheltuieli de personal (Salarii, contribuții)
- **TITLU 20** — Bunuri și servicii (Utilități, reparații etc.)
- **TITLU 71** — Investiții de capital (Construcții, echipamente)
- **TITLU 81** — Rambursări de credite
