---

## Introduction

### Accounting Is the Memory of Compliance

Proper accounting isn't just about numbers—it's the documentation that proves your compliance. When auditors come, your accounts tell the story of every lei spent.

> **Key Insight:** Document retention was reduced to **5 years** by Legea 36/2023 (amending Legea 82/1991). The previous 10-year requirement no longer applies to general financial documents—though financial statements (bilanț) still require 10-year retention.

## Core Concepts

### Public Sector Accounting Framework

```
┌─────────────────────────────────────────────────────────────────────────┐
│                PUBLIC SECTOR ACCOUNTING                                 │
│                (Legea 82/1991)                                          │
│                                                                         │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │  CONTABILITATE DE ANGAJAMENTE (Commitment Accounting)           │  │
│   │  • Records when legal obligation arises                         │  │
│   │  • Regardless of payment timing                                 │  │
│   │  • Tracks against commitment authority                          │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │  CONTABILITATE BUGETARĂ (Budget Accounting)                     │  │
│   │  • Tracks against approved appropriations                       │  │
│   │  • Execution vs. plan comparison                                │  │
│   │  • Classification-based recording                               │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
│   ┌─────────────────────────────────────────────────────────────────┐  │
│   │  CONTABILITATE PATRIMONIALĂ (Asset Accounting)                  │  │
│   │  • Assets, liabilities, equity                                  │  │
│   │  • Accrual-based (IPSAS influenced)                             │  │
│   │  • Full balance sheet accounting                                │  │
│   └─────────────────────────────────────────────────────────────────┘  │
│                                                                         │
└─────────────────────────────────────────────────────────────────────────┘
```

### Standard Execution Report Structure

| Section | Content |
|---------|---------|
| Prevederi inițiale | Initial appropriation |
| Prevederi definitive | After rectifications |
| Angajamente bugetare | Budget commitments |
| Angajamente legale | Legal commitments (signed contracts) |
| Plăți efectuate | Payments made |
| Diferențe și explicații | Variances and explanations |

## Key Takeaways

```
┌─────────────────────────────────────────────────────────────────────────┐
│                          KEY TAKEAWAYS                                  │
│                                                                         │
│   ✓ Document retention reduced to 5 YEARS by Legea 36/2023            │
│     (exception: financial statements 10 years, personnel 75 years)    │
│                                                                         │
│   ✓ Local budget annual accounts due MAY 31 per Legea 273/2006        │
│     (state budget entities: April 30)                                  │
│                                                                         │
```
