## Introduction

### The Legal Architecture of Public Finance

Romanian public finance operates within a complex but structured legal framework. From the Constitution down to ministerial orders, each level of legislation defines and constrains how you manage public money.

> **Key Insight:** When legal requirements conflict, you must apply them in hierarchy order: Constitution first, then EU regulations, organic laws, ordinary laws, and so on. Knowing this hierarchy protects you from liability.

```typescript
interface ConflictScenario {

  ruleB: LegalReference

  result: LegalReference
}

interface ConflictResolver {
  analyze: (ruleA: LegalReference, ruleB: LegalReference) => ConflictScenario
  getHierarchyLevel: (law: string) => number
  isLater: (lawA: string, lawB: string) => boolean
  isMoreSpecific: (lawA: string, lawB: string) => boolean
}
```

### The 2013 Amendments (Legea 270/2013)

Important changes to Legea 500/2002 that updated the budget calendar:

| Original (2002) | Updated (2013) |
|-----------------|----------------|
| Various MF deadlines | MF framework letter: August 1 |
| October proposals | OPC proposals: September 1-15 |
| November submission | Government to Parliament: November 15 |
| — | MF consolidation: September 30 |
