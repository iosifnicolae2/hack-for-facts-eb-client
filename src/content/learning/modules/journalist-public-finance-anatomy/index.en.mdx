## Introduction

### Following the Money Flow

> **Key Insight:** To investigate public money, you must first understand how it flows. Romania's budget system is like a river with many tributaries â€” learning the map is essential.

Romania's public finances involve approximately 40-44% of GDP flowing through multiple interconnected budgets. Understanding this structure is your foundation for every investigation.

## Technical Notes for Implementation

### Components Needed

1. **BudgetStructureExplorer** - Interactive hierarchical diagram
2. **ActorCardDeck** - Swipeable card component
3. **DeficitCalculator** - Interactive slider with live calculation
4. **MoneyFlowDiagram** - Animated flow visualization
5. **UATStructure** - Collapsible hierarchy view
6. **MapIntegration** - Embedded map with highlights

### Data Requirements

- Current GDP figure (static, updated annually)
- Spending/Revenue as % of GDP (static, updated quarterly)
- EU comparison data (static, updated annually)
- County and UAT counts (static: 41+1, ~3,229)
- Entity spending data for comparisons (API)

### API Endpoints

- `GET /api/map/counties` - County-level spending
- `GET /api/map/uats?county={id}` - UAT-level spending
- `GET /api/entities/compare` - Entity comparison data
- `GET /api/analytics/fiscal-position` - National fiscal metrics

### Accessibility

- Sliders have keyboard controls and ARIA labels
- Card deck navigable with arrow keys
- Diagrams have text descriptions
- Color coding supplemented with patterns/labels
