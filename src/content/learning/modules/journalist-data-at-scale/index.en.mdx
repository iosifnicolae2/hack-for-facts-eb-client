## Learning Objectives

By the end of this module, you will be able to:

- [ ] Handle different data formats (PDF, Excel, CSV, API)
- [ ] Clean and normalize budget data for analysis
- [ ] Write basic SQL queries for anomaly detection
- [ ] Use tools for data visualization and exploration

## Introduction

### From Spreadsheets to Stories

> **Key Insight:** SQL skills multiply your investigation capacity 100x. Instead of manually comparing 10 entities, you can analyze 3,000 in seconds.

Budget data comes in many formats and sizes. Mastering data handling transforms you from a reporter who can analyze one entity to an investigator who can find patterns across thousands.
