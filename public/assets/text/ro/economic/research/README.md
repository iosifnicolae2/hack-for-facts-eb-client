# Prompturi de Cercetare pentru Clasificația Economică

Acest folder conține prompturi de cercetare pentru fiecare capitol din clasificația economică a cheltuielilor bugetare conform Ordinului MFP nr. 1954/2005.

## Cum se folosesc

Fiecare fișier conține un prompt complet în limba română care poate fi copiat și folosit cu un model AI (ChatGPT, Claude, etc.) pentru a genera documentația tehnică detaliată pentru capitolul respectiv.
